一、什么是 Prompt 工程

Prompt Engineering（提示工程）是指：

通过设计和优化输入给大语言模型（LLM）的提示（Prompt），从而获得更准确、更稳定输出的一种技术。

在使用大语言模型（如 GPT、Claude 等）时，模型输出不仅取决于模型本身能力，还取决于 Prompt 的设计质量。

可以简单理解为：

Prompt 工程的核心目标：

  • 明确任务
  • 提供必要上下文
  • 规范输出结构
  • 引导模型推理

本质上可以理解为：

用自然语言设计一种可控的“AI程序接口”。

Prompt 的基本结构

一个高质量 Prompt 通常包含四个部分：

组成部分 作用
Role（角色） 设定 AI 的身份或专业领域
Task（任务） 明确需要完成的任务
Context（上下文） 提供相关信息
Output Format（输出格式） 限制输出结构

常见 Prompt 技术

1 Zero-shot Prompt

不提供示例，直接给任务。

特点：

  • 简单
  • 适用于基础任务

2 Few-shot Prompt

提供示例让模型模仿。

优点：

  • 提高准确率
  • 控制输出风格

3 Chain-of-Thought（CoT）

引导模型逐步推理。

适用于：

  • 数学问题
  • 逻辑推理
  • 复杂任务
