流体的连续介质假设

流体的定义

物质的形态

  • 固态、固体

  • 液态、液体、流体

  • 气态、气体、流体

  • 固体

    • 能抵抗一定数量的拉力，压力和剪切力

  • 流体

    • 不能承受拉力静止流体不能承受剪切力，没有固定的形状，易于流动
    • 定义：在微小剪切力的持续作用下能够连续变形的物质

流体的也正

  • 液体
    • 不易压缩
    • 具有一定体积，有自由表面
  • 气体
    • 易压缩
    • 没有固定形状，没有自由表面
  • 微观
    • 流体由分子构成，分子间存有空隙，在空间是不连续的
  • 宏观
    • 一般工程中所研究流体的空间尺度要比分子距离大得多，关心的是宏观流动问题

流体的连续介质假设内容

以流体微团（或流体质点）最小单位，不考虑流体分子间的间隙，把流体视为由无数连续分布的流体微团组成的连续介质

  • 宏观上无限小
  • 微观上足够大
  1. 必要性
    • 在连续介质假说下，物理量在液体中连续分布，将流体的各种物理量看作是空间坐标和时间的连续函数，从而能够使用解析方法等数学工具来研究流体的平衡和运动规律
  2. 合理性
    • 工程上并不关心个别分子的运动，关心的是无数个流体微团所组成宏观流体的连续流动，这样假设的结果与实际相差不大
  3. 适用范围

    • 物体的特征尺寸远远大于流体质点特征尺寸

      \[ \frac{L}{l}\ge 100 \]

      • \(L\)：物体的特征尺寸
      • \(l\)：流体质点的特征尺寸

流体的主要物理性质

密度

  • 定义
    • 单位体积流体的质量 \(\rho\)

      • 均质流体

        \[ \rho=\frac{m}{V} \]

      • 非均质流体

        \[ \rho=\lim_{\Delta V\to 0}\frac{\Delta m}{\Delta V} \]

  • 相对密度

    • 流体密度与水密度的比值

      \[ d=\frac{\rho_f}{\rho_w} \]

  • 影响因素
    • 种类
    • 温度
    • 压强
  • 水密度的变化
    • 查表得到
  • 气体密度的变化

    • 完全气体的状态方程

      \[ \rho_2=\rho_1\frac{p_2T_1}{p_1T_2} \]

      • 通过理想状态方程 \(\displaystyle pV=nRT\) 推导得到

压缩性与膨胀性

流体的压缩性

温度 \(T\) 一定，\(P\) 增大导致 \(V\) 减小

体积压缩系数 \(\beta\) 为温度保持不变时单位压强增量引起的流体体积减小量

\[ \beta = -\frac{1}{V}\frac{\mathrm{d}V}{\mathrm{d}P} \]

流体的膨胀性

\(P\) 一定，\(T\) 增大导致 \(V\) 增大

体积膨胀系数 \(\alpha_V\) 为压强不变时单位温度增量引起的流体体积相对增大量

\[ \alpha_V = \frac{1}{V}\frac{\mathrm{d}V}{\mathrm{d}T} \]

可压缩流体和不可压缩流体
  • 不可压缩流体

    • 液体密度随温度和压强的变化很小，可以近似看作常数

      \[ \frac{\mathrm{d}\rho}{\mathrm{d}t}=0 \]

      \[ \rho = \text{const.} \]

  • 可压缩流体

    • 气体密度随温度和压强的变化较大，不能近似看作常数

      \[ \frac{\mathrm{d}\rho}{\mathrm{d}t}\ne 0 \]

      \[ \rho \ne \text{const.} \]

  • 严格地说，不存在完全不可压缩的流体
  • 一般情况下，把液体都视为不可压缩流体，气体视为可压缩流体
  • 对于液体，管路中压强变化较大时，必须作为可压缩流体（发生水击、水下爆破）
  • 对于气体，当所受压强变化相对较小时，也可视为不可压缩流体

黏性与牛顿内摩擦定律

黏性
  • 流体微团间发生相对滑移时产生切向阻力的性质
  • 流体内部各流体微团之间会产生黏性力
  • 流体将黏附于它所接触的固体表面
牛顿内摩擦定律

内摩擦力 \(F\)

  • 与垂直于流动方向的速度梯度 \(\displaystyle \frac{\mathrm{d}u}{\mathrm{d}y}\) 成正比
  • 与流体接触面积 \(A\) 成正比
  • 与流体种类有关
  • 与接触面压强 \(P\) 无关

\[ F \propto A \frac{\mathrm{d}u}{\mathrm{d}y} \]

\[ F= \mu A \frac{\mathrm{d}u}{\mathrm{d}y} \]

\(\tau\) 表示流层间单位面积上的内摩擦力，即 切向应力

\[ \tau = \frac{F}{A} = \mu \frac{\mathrm{d}u}{\mathrm{d}y} \]

此为牛顿内摩擦定律

  • \(\mu\) ：动力黏度，单位为 \(\mathrm{Pa}\cdot\mathrm{s}\)，反映了液体粘滞性大小的程度
  • \(v\) ：运动黏度，单位为 \(\mathrm{m}^2/\mathrm{s}\)，定义为 \(\displaystyle v=\frac{\mu}{\rho}\)，反映了流体流动阻力大小的程度
影响黏性的因素
  • 流体种类 相同条件下，液体的黏性大于气体的黏性
  • 压强
    • 常压下压强对流体的黏性影响很小，可以忽略不计
    • 高压下流体的黏性随压强的增大而增大
  • 温度
    • 液体的黏性随温度的升高而降低
    • 气体的黏性随温度的升高而增大
牛顿流体和非牛顿流体

作用在流体上的力

表面力

  • 作用在流体中某部分流体体积表面上的力，也就是相邻流体或固体通过接触面作用在其上的力
  • 大小与作用面积成正比

\[ \Delta F = \Delta P + \Delta T \]

  • 法向力 \(\Delta P\)

    • 法向应力

      \[ p = \lim_{\Delta A\to 0}\frac{\Delta P}{\Delta A}=\frac{\mathrm{d}P}{\mathrm{d}A} \]

  • 切向力 \(\Delta T\)

    • 切向应力

      \[ \tau = \lim_{\Delta A\to 0}\frac{\Delta T}{\Delta A}=\frac{\mathrm{d}T}{\mathrm{d}A} \]

质量力

  • 作用在流体某体积内所有流体质点上并与这一体积的流体质量成正比的力，又称体积力
  • 非接触力，与流体的质量成正比

\[ \vec{f}=\frac{\vec{F}}{m}=f_x\vec{i}+f_y\vec{j}+f_z\vec{k} \]

表面张力与毛细现象

表面张力

  • 在液体自由表面层由于分子引力作用而产生的使液体表面层收缩的力

  • 处于两种不同的流体介质分界面上的分子所受到不平衡的分子吸引力的宏观表象

  • 产生位置： 存在于液体与气体、液体与固体 或 两种不同液体的分界面上

  • 产生原因： 分子在界面处所受的引力不同

  • 作用方向： 表面张力的方向始终与自由界面相切

毛细现象

液体在细管中能上升或下降的现象

  • 内聚力 分子间的相互吸引力（指同种物质内部）。
  • 附着力： 指两种不同物质接触部分的相互吸引力（例如液体与管壁固体之间）。

流体静力学基础

流体平衡应力特征与静压强

流体平衡应力特征

  • 没有黏性切向力
  • 只有法向分力
  • 不能承受拉力
  • 压力是唯一表面力

静压强基本特征

  1. 流体静压强方向与作用面相垂直，并指向作用面的内法线方向（方向）
  2. 静止流体中任意一点流体压强的大小与作用面的方位无关，即任一点上各方向的流体静压强都相同（大小）

    • 流体的动压强定义为三个互相垂直的坐标轴上流体静压强的平均值

      \[ p=\frac{p_x+p_y+p_z}{3} \]

流体平衡微分方程

流体平衡微分方程的推导

在静止流体中去一平行六面体微团，边长为 \(\mathrm{d}x\)、\(\mathrm{d}y\)、\(\mathrm{d}z\)，体积为 \(\mathrm{d}V\)，质量为 \(\mathrm{d}m\)，受质量力 \(\vec{f}\) 和表面力 \(\vec{p}\) 的作用

取 \(x\) 方向进行分析

  • 质量力

    \[ f_x\mathrm{d}x\mathrm{d}y\mathrm{d}z \]

  • 表面力

    \[ p(x+\frac{\mathrm{d}x}{2})\mathrm{d}y\mathrm{d}z - p(x-\frac{\mathrm{d}x}{2})\mathrm{d}y\mathrm{d}z = \frac{\partial p}{\partial x}\mathrm{d}x\mathrm{d}y\mathrm{d}z \]

根据平衡条件，\(x\) 方向的质量力与表面力平衡

\[ \begin{aligned} p(x+\frac{\mathrm{d}x}{2})\mathrm{d}y\mathrm{d}z - p(x-\frac{\mathrm{d}x}{2})\mathrm{d}y\mathrm{d}z - f_x\mathrm{d}x\mathrm{d}y\mathrm{d}z &= 0 \\ \frac{\partial p}{\partial x}\mathrm{d}x\mathrm{d}y\mathrm{d}z - f_x\mathrm{d}x\mathrm{d}y\mathrm{d}z &= 0 \\ p_x - f_x &= 0 \end{aligned} \]

同样的，在 \(y\) 和 \(z\) 方向上也可以得到类似的表达式

\[ \left\{ \begin{aligned} &f_x - \frac{1}{\rho}\frac{\partial p}{\partial x} = 0 \\ &f_y - \frac{1}{\rho}\frac{\partial p}{\partial y} = 0 \\ &f_z - \frac{1}{\rho}\frac{\partial p}{\partial z} = 0 \end{aligned} \right. \]

写成矢量形式

\[ \vec{f}-\frac{1}{\rho}\nabla p = 0 \]

压强差公式

\[ \rho(f_x\mathrm{d}x+f_y\mathrm{d}y+f_z\mathrm{d}z) = \frac{\partial p}{\partial x}\mathrm{d}x+\frac{\partial p}{\partial y}\mathrm{d}y+\frac{\partial p}{\partial z}\mathrm{d}z \]

即

\[ \mathrm{d}p = \rho(f_x\mathrm{d}x+f_y\mathrm{d}y+f_z\mathrm{d}z)= \rho\vec{f}\cdot\mathrm{d}\vec{r} \]

流体平衡条件

旋度为 \(0\)

\[ \nabla \times \vec{f} = 0 \]

展开

\[ \left\{ \begin{aligned} &\frac{\partial f_z}{\partial y} - \frac{\partial f_y}{\partial z} = 0 \\ &\frac{\partial f_x}{\partial z} - \frac{\partial f_z}{\partial x} = 0 \\ &\frac{\partial f_y}{\partial x} - \frac{\partial f_x}{\partial y} = 0 \end{aligned} \right. \]

即在流体中，质量力场必须是无旋场，才能使流体处于平衡状态

计单位质量体积力的势函数为 \(\phi\)，则

\[ \mathrm{d}p = \rho(f_x\mathrm{d}x+f_y\mathrm{d}y+f_z\mathrm{d}z) = -\rho\left(\frac{\partial \phi}{\partial x}\mathrm{d}x+\frac{\partial \phi}{\partial y}\mathrm{d}y+\frac{\partial \phi}{\partial z}\mathrm{d}z\right) = -\rho\mathrm{d}\phi \]

当 \(\rho\) 是常数时，积分得到

\[ p=-\rho\phi + C \]

流体静力学基本方程

静力学基本方程式的推导

在重力场中，质量力 \(\vec{f}\) 的分量为

\[ \left\{ \begin{aligned} &f_x = 0 \\ &f_y = 0 \\ &f_z = -g \end{aligned} \right. \]

带入流体平衡方程

\[ \mathrm{d}p = \rho(f_x\mathrm{d}x+f_y\mathrm{d}y+f_z\mathrm{d}z) = -\rho g \mathrm{d}z \]

当 \(\rho\) 是常数时，积分得到流体静力学基本方程

\[ \boxed{z+\frac{p}{\rho g} = c} \]

静力学基本方程的物理意义

表示在重力作用下静止流体中各点的单位重量流体的总势能是相等的

  • \(z\)：位置势能
  • \(\displaystyle \frac{p}{\rho g}\)：压强势能
  • \(c\)：常数，表示单位重量流体的总势能

或称在重力作用下静止流体中各点的静水头是相等的

  • 单位重量流体具有的能量用液柱高度来表示称为水头

静力学基本方程的另一种形式

\[ p=p_0 + \rho gh \]

  • \(p_0\)：参考点的压强
  • \(h\)：参考点与所求点之间的垂直距离 \(\displaystyle h=z_0-z\)

压强的测量

压强的测量

静止流体对物体表面的压强合力

平面静水总压力

总压力大小的推导

任意微元面积受到的压力为

\[ \mathrm{d}P=p \mathrm{d}A= \rho g z \sin\alpha \mathrm{d}A \]

积分得到静止液体作用在整个平面上的总压力

\[ P=\iint_{A}\rho gz\sin\alpha \, \mathrm{d}A = \rho g \sin\alpha \iint_{A} z \, \mathrm{d}A \]

记 \(\displaystyle \iint_A z \, \mathrm{d}A\) 为 \(A\) 对 \(OX\) 轴的面积矩，记为 \(z_c\)，则

\[ P=\rho g \sin\alpha z_c A \]

记 \(h_c\) 为 \(z_c \sin\alpha\)，称为形心淹深

\[ P=\rho g h_c A \]

即压强合力等于受压面形心处的压强与面积的乘积，压强合力垂直于平面壁

力的作用点

使用和力矩来求作用点

\[ z_d P=\iint_A z\mathrm{d}P = \iint_A z \rho g z \sin\alpha \mathrm{d}A \]

\[ \iint_A z\mathrm{d}P = \iint_A z \rho g z \sin\alpha \mathrm{d}A=\rho g \sin\alpha \iint_A z^2 \mathrm{d}A \]

定义

\[ J_x = \iint_A z^2 \mathrm{d}A \]

将 \(\displaystyle \iint_A z\mathrm{d}P\) 和 \(P\) 的表达式代入力矩平衡方程

\[ z_d \rho g z_c \sin\alpha = \rho g \sin\alpha J_x \]

借得

\[ z_d = \frac{J_x}{z_c A} \]

使用平行轴定理对这个结论进一步进行化简

\[ z_d = \frac{J_{cx}+{z_c}^2A}{z_c A} = z_c+\frac{J_{cx}}{z_c A} \]

曲面静水总压力

总压力大小推导

可以将力分解为沿 \(x\) 和 \(z\) 方向的两个分力

\[ \mathrm{d}P_x = \rho g h\cos\alpha\mathrm{d}A = \rho g h\mathrm{d}A_x \]

\[ P_x = \iint_A \rho g h \,\mathrm{d}A_x=\rho g h_c A_x \]

\[ \mathrm{d}P_z = \rho g h\sin\alpha\mathrm{d}A = \rho g h\mathrm{d}A_z \]

\[ P_z= \iint_A \rho g h \,\mathrm{d}A_z=\rho g V_p \]

静止液体作用在曲面上的总压力的垂直分力等于压力体的液体重量

总压力作用点

作出 \(P_x\) 及 \(P_z\) 的作用线，得交点，过此交点按倾斜角 \(\alpha\) 作总压力 \(P\) 的作用线，与曲面壁 AB 相交的点，即为总压力 \(\vec{P}\) 的作用点。

压力体定义以及组成
  • 定义：x所研究的曲面（淹没在静止液体中的部分）到自由液面或自由液面的延长面间投影所包围的一块空间体积

压力体体积计算公式

\[ V_p=A_{abcd}\cdot b \]

  1. 受压区面本身
  2. 通过曲面边缘所作的铅垂面
  3. 自由液面或自由液面的延长线
压力体种类
  • 实压力体
    • 方向向下
  • 虚压力体
    • 方向向上

流体动力学基础

描述流体运动的两种方法

拉格朗日法

跟踪每个流体质点的运动全过程，记录运动过程中各物理量及其变化规律

某流体质点位移

\[ \left\{ \begin{aligned} &x=x(a,b,c,t)\\ &y=y(a,b,c,t)\\ &z=z(a,b,c,t) \end{aligned} \right. \]

质点速度

\[ \left\{ \begin{aligned} &u=\frac{\partial x}{\partial t} = \frac{\partial x(a,b,c,t)}{\partial t}\\ &v=\frac{\partial y}{\partial t} = \frac{\partial y(a,b,c,t)}{\partial t}\\ &w=\frac{\partial z}{\partial t} = \frac{\partial z(a,b,c,t)}{\partial t} \end{aligned} \right. \]

质点加速度

\[ \left\{ \begin{aligned} &a_x=\frac{\partial ^2 x}{\partial t^2}\\ &a_y=\frac{\partial ^2 y}{\partial t^2}\\ &a_z=\frac{\partial ^2 z}{\partial t^2} \end{aligned} \right. \]

欧拉法

分析流场中每一空间点上的流体质点运动来研究整个流体的运动规律。着眼于整个流场的状态

\[ \left\{ \begin{aligned} &u=u(x,y,z,t)\\ &v=v(x,y,z,t)\\ &w=w(x,y,z,t)\\ &p=p(x,y,z,t)\\ &\rho=\rho(x,y,z,t) \end{aligned} \right. \]

质点的速度和加速度

\[ a_x=\frac{\mathrm{d}u}{\mathrm{d}t}=\frac{\partial u}{\partial t}+\frac{\partial u}{\partial x}\frac{\mathrm{d}x}{\mathrm{d}t}+\frac{\partial u}{\partial y}\frac{\mathrm{d}y}{\mathrm{d}t}+\frac{\partial u}{\partial z}\frac{\mathrm{d}z}{\mathrm{d}t} \]

\[ \left\{ \begin{aligned} &a_x=\frac{\partial u}{\partial t}+u\frac{\partial u}{\partial x}+v\frac{\partial u}{\partial y}+w\frac{\partial u}{\partial z}\\ &a_y=\frac{\partial v}{\partial t}+u\frac{\partial v}{\partial x}+v\frac{\partial v}{\partial y}+w\frac{\partial v}{\partial z}\\ &a_z=\frac{\partial w}{\partial t}+u\frac{\partial w}{\partial x}+v\frac{\partial w}{\partial y}+w\frac{\partial w}{\partial z} \end{aligned} \right. \]

矢量形式为

\[ \vec{a}=\frac{\mathrm{d}\vec{V}}{\mathrm{d}t}=\frac{\partial \vec{V}}{\partial t}+(\vec{V}\cdot\nabla)\vec{V} \]

流体运动的基本概念

定常运动和非定常运动

流体质点的运动要素只是坐标的函数而与时间无关被称为定常运动

定常运动质点加速度为

\[ \vec{a}=(\vec{V}\cdot\nabla)\vec{V} \]

非定常运动质点加速度为

\[ \vec{a}=\frac{\partial \vec{V}}{\partial t}+(\vec{V}\cdot\nabla)\vec{V} \]

迹线和流线

迹线

在流场中某一流体质点的运动轨迹

为同一流体致电在不同时刻形成的曲线

\[ \left\{ \begin{aligned} &u\mathrm{d}t=\mathrm{d}x\\ &v\mathrm{d}t=\mathrm{d}y\\ &w\mathrm{d}t=\mathrm{d}z \end{aligned} \right. \]

流线

某一瞬时在流场中作的一条曲线，在该曲线上的所有流体质点速度方向都与该曲线相切

同一时刻由不同流体质点组成的曲线

\[ \vec{V}=u\vec{i}+v\vec{j}+w\vec{k} \]

\[ \mathrm{d}s=\mathrm{d}x\vec{i}+\mathrm{d}y\vec{j}+\mathrm{d}z\vec{k} \]

其速度与流线相切则

\[ \vec{v}\times\mathrm{d}\vec{s}= \begin{vmatrix} i & j & k \\ u & v & w \\ \mathrm{d}x & \mathrm{d}y & \mathrm{d}z \end{vmatrix} =0 \]

即

\[ \left\{ \begin{aligned} &u\mathrm{d}y - v\mathrm{d}x = 0 \\ &v\mathrm{d}z - w\mathrm{d}x =0\\ &w\mathrm{d}x - u\mathrm{d}z = 0 \end{aligned} \right. \]

或

\[ \frac{\mathrm{d}x}{u}=\frac{\mathrm{d}y}{v}=\frac{\mathrm{d}z}{w} \]

流管、流束和总流

随着速度的增大，流体运动状态由层流转变为紊流，当达到特定的区间 \((v_c，v_c')\) 时处于过渡状态

  • 上临界流速 \(v_c'\)：过渡状态转变为紊流的速度

    • 管径大的临界流速小

雷诺数

\[ V_c \propto \frac{\mu}{\rho d}=\mathrm{Re}_c \]

  • \(\mathrm{Re}\)：雷诺数，为无量纲数
  • \(\mathrm{Re}_c\)：下临界雷诺数，表示流体由层流转变为紊流的临界状态，大约为 \(2300\)
    • \(\mathrm{Re}>\mathrm{Re}_c\)：紊流
    • \(\mathrm{Re}<\mathrm{Re}_c\)：层流
  • \(\mathrm{Re}_c'\) ：上临界雷诺数，易受外界干扰，数值不稳定
求雷诺数

\[ \mathrm{Re}=\frac{Vd_e}{\nu} \]

  • \(d_e\) 为当量直径

    \[ d_e = \frac{4A}{\kappa} \]

    • \(\kappa\)：湿周，即流体与固体接触的周长
物理意义

\[ \mathrm{Re}=\frac{\rho Vl}{\mu}=\frac{\rho V^2l^2}{\mu Vl}=\frac{\text{惯性力}}{\text{粘性力}} \]

  • 雷诺数表示流体惯性力与粘性力的比值
  • 雷诺数越大，惯性力越占优势，流体越容易发生紊流
  • 雷诺数越小，粘性力越占优势，流体越容易保持层流状态

粘性流体的两种流动状态

  • 层流
    • 流体质点不相互混杂，流体作有序的成层流动
      • 有序性
      • 黏性占主要作用，遵循牛顿内摩擦定律
      • 能量损失与流速的一次方成正比
      • 在流速较小且雷诺数 \(\mathrm{Re}\) 较小时发生
  • 紊流
    • 速度、压力等物理量在时间和空间中发生不规则脉动的流体运动
      • 无序性、随机性、有旋性、混掺性
      • 紊流受黏性和紊动的共同作用
      • 水头损失与流速的 \(1.75\sim 2\) 次方成正比
      • 在流速较大且雷诺数较大时发生

沿程阻力和沿程损失

  • 沿程阻力
    • 流体在管道中流动时，受到管道内壁的摩擦作用而产生的阻力
  • 沿程损失
    • 沿程阻力引起的能量损失，表现为水头损失

达西-威斯巴赫公式

\[ \left\{ \begin{aligned} &h_f = \frac{\lambda l}{d}\frac{V^2}{2g} \\ &\Delta p_f=\lambda \frac{l}{d}\rho \frac{V^2}{2} \end{aligned} \right. \]

  • \(h_f\)：沿程损失
  • \(\Delta p_f\)：沿程压强损失
  • \(\lambda\)：达西摩擦系数，取决于流动状态和管道内壁的粗糙程度
    • 层流：\(\lambda = \frac{64}{\mathrm{Re}}\)
    • 紊流：\(\lambda\) 的数值较大，且与雷诺数的关系较为复杂，通常通过经验公式或图表来确定
  • \(l\)：管道长度
  • \(d\)：管道内径
  • \(V\)：流体平均流速

局部阻力和局部损失

  • 局部阻力
    • 流体在管道中流动时，遇到管道内的局部结构（如弯头、阀门、收缩或扩张等）时产生的阻力
  • 局部损失
    • 局部阻力引起的能量损失，表现为水头损失

\[ \left\{ \begin{aligned} &h_j = \zeta \frac{V^2}{2g}\\ &\Delta p_f = \zeta \rho \frac{V^2}{2} \end{aligned} \right. \]

热量传递的三种基本方式
  • 热传导

    • 满足傅立叶定律

      \[ \frac{\Phi}{A}=q=-\lambda \frac{\mathrm{d}t}{\mathrm{d}x} \]

  • 热对流

    • 流体各部分发生相对位移时，冷热流体相互掺混所引起的热量传递过程

    • 对流换热满足牛顿冷却公式

      \[ \left\{ \begin{aligned} &q=\frac{\Phi}{A}=h(t_w-t_f)\ t_w>t_f\\ &q=\frac{\Phi}{A}=h(t_f-t_w)\ t_w<t_f \end{aligned} \right. \]

  • 热辐射

    • 物体由于热的原因向外发射的辐射

    • 黑体辐射满足斯特藩-玻尔兹曼定律

      \[ \Phi=A\sigma t^4 \]

    • 实际物体热辐射满足

      \[ \Phi = \varepsilon A \sigma t^4 \]

热阻思想

\[ \Phi = \lambda A\frac{\Delta t}{\delta}=\frac{\Delta t}{\delta/(\lambda A)} \]

  • \(\delta/(\lambda A)\)：热阻，表示热量传递的难易程度
  • \(\Delta t\)：温压

这样我们就将传热学和电学相联系起来了

一般传热过程可以看作有三个串联环节

  1. 高温流体与固体壁面的换热
  2. 通过壁面的导热
  3. 低温流体与固体另一面的换热

\[ \left\{ \begin{aligned} &\Phi = Ah_1(t_{f_1}-t_{w1}) \rightarrow \Phi = \frac{t_{f1}-t_{w1}}{1/(h_1A)}\\ &\Phi = \lambda A\frac{t_{w1}-t_{w2}}{\delta} \rightarrow \Phi = \frac{t_{w1}-t_{w2}}{\delta/(\lambda A)}\\ &\Phi = Ah_2(t_{w2}-t_{f2}) \rightarrow \Phi = \frac{t_{w2}-t_{f2}}{1/(h_2A)} \end{aligned} \right. \]

通过热阻思想可以写成

\[ \Phi = \frac{t_{f1}-t_{f2}}{\frac{1}{h_1A}+\frac{\delta}{\lambda A}+\frac{1}{h_2A}} \]

\[ \lim_{\Delta n \to 0}\left(\frac{\Delta t}{\Delta n}\right)\vec{n} = \frac{\partial t}{\partial n}\vec{n}= \nabla t = \frac{\partial t}{\partial x}+\frac{\partial t}{\partial y}+\frac{\partial t}{\partial z} \]

导热的理论基础与计算

傅里叶基本定律

傅里叶定律

\[ \vec{q}=\frac{\vec{\phi}}{A}=-\lambda \nabla t=-\lambda \frac{\partial t}{\partial n}\vec{n} \]

可以将热流密度矢量分解为三个坐标轴上的分量

\[ \left\{ \begin{aligned} &q_x=-\lambda \frac{\partial t}{\partial x}\\ &q_y=-\lambda \frac{\partial t}{\partial y}\\ &q_z=-\lambda \frac{\partial t}{\partial z} \end{aligned} \right. \]

  • 负号说明热量传递方向指向温度降度方向
  • 热流密度矢量与等温线或等温面垂直，向与温度梯度相反

导热系数

\[ \lambda=\frac{\vert q\vert}{\vert \nabla t \vert} \]

数值上等于单位温度梯度作用下的物体内产生的热流密度的模

我们将工程导热材料分为四类

  • 均匀，各向同性
  • 均匀，各向异性
  • 非均匀，各向同性
  • 非均匀，各向异性

在温度变化范围不是很大的情况下，可以将工程导热材料的导热系数表示为温度的线性函数

\[ \lambda = \lambda_0(1+\alpha t) \]

导热微分方程及定解条件

导热微分方程的推导

  1. 单位时间内导入微元体的净热量

    \[ \Phi_c = \left[\frac{\partial }{\partial x}\left(\lambda \frac{\partial t}{\partial x}\right)+\frac{\partial }{\partial y}\left(\lambda \frac{\partial t}{\partial y}\right)+\frac{\partial }{\partial z}\left(\lambda \frac{\partial t}{\partial z}\right)\right]\,\mathrm{d}x\mathrm{d}y\mathrm{d}z \]

  2. 单位时间内微元体内热源生成的热量

    \[ \Phi_V = \dot{\Phi}\, \mathrm{d}x\mathrm{d}y\mathrm{d}z \]

  3. 单位时间内微元体热力学能的增量

    \[ \Delta U =\rho c \frac{\partial t}{\partial \tau}\cdot \mathrm{d}x\mathrm{d}y\mathrm{d}z \]

根据热量守恒定律，导热微分方程为

\[ \underbrace{\rho c \frac{\partial t}{\partial \tau}}_{\begin{array}{c} \text{非稳态项} \\ \text{(Transient term)} \\ \text{热力学能增量} \end{array}} = \underbrace{\frac{\partial}{\partial x} \left( \lambda \frac{\partial t}{\partial x} \right) + \frac{\partial}{\partial y} \left( \lambda \frac{\partial t}{\partial y} \right) + \frac{\partial}{\partial z} \left( \lambda \frac{\partial t}{\partial z} \right)}_{\begin{array}{c} \text{导热项（扩散项）} \\ \text{(Diffusion term)} \\ \text{三个坐标方向导入的净热量} \end{array}} + \underbrace{\dot{\Phi}}_{\begin{array}{c} \text{内热源项} \\ \text{(Source term)} \end{array}} \]

导热微分方程的简化形式

\(\lambda\) 为常数的情况

\[ \frac{\partial t}{\partial \tau}=a \left(\frac{\partial^2 t}{\partial x^2}+\frac{\partial^2 t}{\partial y^2}+\frac{\partial^2 t}{\partial z^2}\right)+\frac{\dot{\Phi}}{\rho c} \]

  • \(a\)：热扩散率，\(a = \frac{\lambda}{\rho c}\)
\(\lambda\) 为常数且无内热源

\[ \frac{\partial t}{\partial \tau}=a\Delta t \]

\(\lambda\) 为常数且稳态

\[ \Delta t + \frac{\dot{\Phi}}{\lambda} = 0 \]

被称为 Poisson 方程

\(\lambda\) 为常数、无内热源且稳态

\[ \Delta t = 0 \]

被称为 Laplace 方程

\(\lambda\) 为常数、无内热源且稳态的一维导热微分方程

\[ \frac{\mathrm{d}^2 t}{\mathrm{d} x^2}=0 \]

其他坐标系中的导热微分方程

  • 柱坐标系

    \[ \rho c =\frac{1}{r} \frac{\partial}{\partial r}\left(r \lambda \frac{\partial t}{\partial r}\right)+\frac{1}{r^2} \frac{\partial}{\partial \theta}\left(\lambda \frac{\partial t}{\partial \theta}\right)+\frac{\partial}{\partial z}\left(\lambda \frac{\partial t}{\partial z}\right)+\dot{\Phi}s \]

  • 球坐标系

    \[ \rho c =\frac{1}{r^2} \frac{\partial}{\partial r}\left(r^2 \lambda \frac{\partial t}{\partial r}\right)+\frac{1}{r^2 \sin \theta} \frac{\partial}{\partial \theta}\left(\sin \theta \lambda \frac{\partial t}{\partial \theta}\right)+\frac{1}{r^2 \sin^2 \theta} \frac{\partial}{\partial \varphi}\left(\lambda \frac{\partial t}{\partial \varphi}\right)+\dot{\Phi} \]

非稳态导热的基本概念与数学描述

非稳态导热的分类及特点

分类
  • 周期性
    • 物体中的各点温度以及热流密度随时间作周期性变化
  • 非周期性
    • 物体的温度随时间变化逐渐趋向于稳定
特点
  1. 界面上所发生的热扰动传递到一定的距离需要一定的时间 在同一时刻，热量传递方向不同截面上的导热量不相同
  2. 在非稳态导热物体中的温度分布存在两个不同的阶段
    1. 非正规阶段 温度分布主要受到初始温度分布的影响
    2. 正规阶段 初始温度分布影响影响逐渐消失，不同时刻温度分布主要取决于边界条件以及物性

  2. \(t_w\) 未知的表面温度

第三类边界条件下 Bi 数对非稳态导热的影响

我们将厚为 \(2\delta\) 的平板中非稳态导热情况根据导热物体的导热系数和表面换热条件分成三种情况

  1. 表面换热十分强烈 \[ \frac{1}{h}<<\frac{\delta}{\lambda} \]
  2. 表面换热十分微弱 \[ \frac{1}{h}>>\frac{\delta}{\lambda} \]
  3. 表面换热热阻约等于导热热阻 \[ \frac{1}{h} \approx \frac{\delta}{\lambda} \]

毕渥数 Bi 的定义

\[ \mathrm{Bi} = \frac{\text{内部导热热阻}}{\text{表面传热热阻}} =\frac{\delta/\lambda}{1/h} = \frac{h\delta}{\lambda} \]

  • \(\displaystyle \mathrm{Bi}\to\infty\) 时说明表面传热系数无穷大，对流传热热阻接近 \(0\)，表面温度等于流体温度
  • \(\displaystyle \mathrm{Bi}\to 0\) 时说明表面传热系数接近 \(0\)，对流传热热阻接近 \(\infty\)，任何时间物体内部的温度分布都趋于一致
  • \(\displaystyle \mathrm{Bi}\approx 1\) 时说明表面传热热阻与内部导热热阻相当，物体内部的温度分布不均匀，表面温度与流体温度之间存在一定的差异

类似 Bi 数这样的表征物理过程特征的无量纲数被称为特征数，也叫准则数，特征数定义中的尺度被称为特征尺度，常用 \(l\) 表示

集中参数法解零维问题

集中参数法温度场分析解

当 \(\mathrm{bi}\to 0\) 时物体内部热阻远小于表面传热热阻，任意时刻道题内部的各点温度趋于一致，只随时间变化不随空间变化，属于零维问题，这样的分析法被称为集中参数法

当 \(\displaystyle \mathrm{Bi} =\frac{hl}{\lambda} \ge 0.1\) 时可以使用集中参数法

数学描写

\[ \frac{\partial t}{\partial \tau} = a\nabla^2 t + \frac{\dot{\Phi}}{\rho c} \]

空间导数为 \(0\) 则

\[ \frac{\mathrm{d}t}{\mathrm{d}\tau} = \frac{\dot{\Phi}}{\rho c} \]

结合第三类边界条件可以得到定解问题

\[ \left\{ \begin{aligned} &\frac{\mathrm{d}t}{\mathrm{d}\tau} = \frac{\dot{\Phi}}{\rho c}\\ &\tau = 0, t = t_0\\ &Ah(t-t_f) = -V \dot{\Phi} \end{aligned} \right. \]

解得

\[ \frac{\theta}{\theta_0}=\frac{t-t_\infty}{t_0-t_\infty}=e^{-\frac{hA}{\rho Vc}\tau} \]

其中 \(\theta = t-t_\infty\) 被称为过余温度

可以将指数改写为

\[ \frac{hA}{\rho Vc}\tau=\frac{h(V/A)}{\lambda}\frac{a\tau}{\left(V/A\right)^2}=\mathrm{Bi}_V\mathrm{Fo}_V \]

  • \(\mathrm{Bi}_V\) 为特征尺度 \(l\) 用 \(V/A\) 表示的毕渥数
  • \(\mathrm{Fo}_V\) 为特征尺度 \(l\) 用 \(V/A\) 表示的傅里叶数
    • 为表征非稳态过程进行深度的无量纲时间，其值越大，热扰动越深入传播到物体内部，物体内各点的温度就越接近周围介质的温度
分析解的应用
计算非稳态导热量

\[ \begin{aligned} \Phi(\tau)=&-\rho c V \frac{\mathrm{d}t}{\mathrm{d}\tau} \\ =& (t-t_\infty) \frac{hA}{\rho Vc} =(t_0-t_\infty)hAe^{\left(-\frac{hA}{\rho c V}\tau\right)} \end{aligned} \]

\[ Q_{0-\tau} = \int _0^\tau \, \mathrm{d}\tau \]

或者先积分

\[ \begin{aligned} Q_{0-\tau} = &\int _0^\tau \Phi(\tau) \, \mathrm{d}\tau \\ =& \int _{t_0}^t \, \mathrm{d}t\\ =& \rho V c(t-0-t_\infty+t_\infty -t)\\ =& \theta_0 \rho V c (1-e^{-\frac{hA}{\rho Vc}\tau}) \end{aligned} \]

时间常数

令 $= 1 $ 的时间为 \(\tau_c\) 称为时间常数

此时 \(\displaystyle \frac{theta}{\theta_0} = e^{-1} \approx 0.368\)，过余温度下降到初始过余温度的 \(36.8\%\)，也就是说物体内的温度已经接近周围介质的温度

时间常数越小，对温度变化的响应越快
